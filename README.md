🍪 BercUITM – Reviewer Management System

A web-based system developed during internship to manage ethical application reviews.
The system allows coordinators to assign applications to reviewers and track evaluation results efficiently.

🚀 Key Contributions
Developed reviewer assignment workflow
Implemented structured review form (BERC)
Built review submission & status tracking system
Integrated PDF generation using FPDF
Designed database structure for review storage (JSON-based)
🛠️ Tech Stack
PHP (Native)
MySQL
JavaScript
Bootstrap
FPDF
XAMPP
📌 System Workflow
Coordinator → Assign Reviewer → Pending
Reviewer → Submit Review → Completed
System → Store Data + Generate PDF
⚙️ Setup
git clone https://github.com/your-username/bercuitm.git
Import database (create_reviews_table.sql)
Run using XAMPP (Apache + MySQL)
▶️ Usage
http://localhost/BercUitm/
Coordinator → assign applications
Reviewer → submit evaluation
📄 Output
Review stored in database
PDF generated in /Reviewer/reviews_pdf/
